# SparEnergi.dk API Documentation

This directory contains the complete documentation for the SparEnergi.dk API.

## Files

-   [**`openapi.yaml`**](./openapi.yaml): The formal OpenAPI 3.0 specification for the API.
-   [**`README.md`**](./README.md): A user-friendly guide to the API, including endpoints and examples.
-   [**`workbook.md`**](./workbook.md): A detailed log of the API discovery and mapping process.
-   [**`examples/`**](./examples/): A directory containing code examples in various languages.
    -   [**`bash-examples.sh`**](./examples/bash-examples.sh): Bash script with `curl` examples.
    -   [**`javascript-examples.js`**](./examples/javascript-examples.js): Node.js script with `fetch` examples.
    -   [**`python-examples.py`**](./examples/python-examples.py): Python script with `requests` examples.
